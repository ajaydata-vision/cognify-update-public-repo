# Public Update Repo Rules

This repository is strictly for Cognify update artifacts only.

Allowed files:
- `update.txt`
- compiled/update zip bundles such as `Cognify-app-update-*.zip`
- optional checksum/signature files for those bundles

Forbidden files:
- source code
- `src/`
- scripts
- docs
- package manifests
- development logs
- any app repository files except compiled update artifacts and `update.txt`

Do not push source code or project documentation here.
